# First Kubernetes Pod (Minikube + WSL)

This project demonstrates running a first Kubernetes Pod using **minikube** and **kubectl** inside **WSL (Ubuntu)** on Windows.

## Environment
- OS: Windows + WSL2 (Ubuntu)
- Architecture: amd64
- Kubernetes: minikube
- Driver: Docker
